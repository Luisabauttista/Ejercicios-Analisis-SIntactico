# Ejercicios-Analisis-Sintactico

En este trabajo se implementaron 3 ejercicios de analisis sintanto, desarrollados en Python ejecutados en la terminal de Linux. 

Se trabajan los siguientes conceptos:

Generación de AST (Abstract Syntax Tree)
Implementación del algoritmo CYK
Uso del algoritmo Earley
Análisis de precedencia y asociatividad

Estructura del proyecto
Punto1AST/
1. ejercicio1.py   # Generación de AST
2. ejercicio2.py   # CYK + Earley
3. ejercicio3.py   # Precedencia y asociatividad
4. README

Ejercicio 1: Generación de AST
python Ejercicio1.py
Permite ingresar una expresión matemática y genera su árbol sintáctico (AST).

Ejercicio 2: Algoritmos CYK y Earley
python Ejercicio2.py

Implementación del algoritmo CYK (gramática en CNF)
Parser usando algoritmo Earley

Permite comparar ambos métodos de análisis.

Ejercicio 3: Precedencia y Asociatividad
python Ejercicio3.py

Permite comprobar cómo la gramática define:

Prioridad de operadores
Forma de agrupación de expresiones

Ejemplos de prueba
Precedencia
10+5*2
Asociatividad
10-5-2
